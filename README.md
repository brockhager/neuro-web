# neuro-web

Web application and frontend for the NeuroSwarm project.

## Overview

This Next.js application provides the user interface for interacting with NeuroSwarm, including personal AI management, proposal submission, and network monitoring.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Features

- Personal AI dashboard
- Proposal interface
- Network status
- Wallet integration
